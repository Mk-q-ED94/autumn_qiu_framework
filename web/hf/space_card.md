# 秋 · Autumn

多模型协作工作流框架的 Web 端。WP1 选择/校验 · WP2 任务执行 · WP3 使命 · Mom 记忆区。

打开后进入 **Settings**，填入你自己的 A1 / A2 / A3 模型 API key、base URL 和模型名，
点 **Apply**，即可开始对话。本 Space 不内置任何密钥——每位用户使用自己的 key。

> 想给 Space 加访问口令：在 **Settings → Variables and secrets** 添加 secret
> `AUTUMN_API_TOKEN`，然后在前端 Settings 里填同样的 token 即可。
